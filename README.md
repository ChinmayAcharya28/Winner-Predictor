# Winner-Predictor

This is a simple python script which can be used to predict the winner. This script calculates the winner prediction on the basis of the previous team records. 

### Prerequisite

To proceed further need to have a previous record of both the teams.

### Installation

 1. Install [Python](https://www.python.org/downloads/) with latest version 3.7.
 
 2. Install [Anaconda Navigator](https://docs.anaconda.com/anaconda/install/).
		 
 3. Launch [Jupyter Notebook](https://jupyter.org).
 
Follow these simple steps and you are done with the installation. 

Everyting is ready! Enjoy your game and code!
